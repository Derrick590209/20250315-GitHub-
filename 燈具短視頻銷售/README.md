# 利用短視頻做燈具銷售的五個步驟
## 第一步：定位你的「專業人設」
- 建築照明顧問
- 照明節能專家
- 健康光環境設計師
- 30年照明老司機  
🎯重點:  
讓客戶覺得「這個人懂光」。

例如：

「為什麼你家客廳看起來便宜？」

「燈選錯，空間直接掉價50%」

「90%設計師都忽略的色溫問題」

👉 先賣觀念，再賣產品


## 第二步：設計三種固定影片類型設計三種固定影片類型
***短影音一定要有「欄目化」***  
1.照明錯誤示範型   
例如：  
- 這種客廳燈千萬不要買  
- 24度 vs 60度差在哪？
2.空間改造對比型
  拍「開燈前 vs 開燈後」  
 （這種超吸睛）  
3.專業知識拆解型  
- 什麼是UGR？
- CRI為什麼比瓦數重要？
- 為什麼高演色燈看起來比較貴？  
## 第三步：拍攝公式（超簡單）
***短影音結構公式：
`3秒鉤子 + 30秒價值 + 5秒引導`***  
範例：  
🎬 開場：  
「你家客廳如果用這種燈，空間會變廉價。」  
🎬 中段：  
說明光束角錯誤 → 示範牆面光斑問題  
🎬 結尾：  
「想知道你家適合什麼光？留言『客廳』我教你。」  
👉 一定要有Call To Action。  

## 第四步：建立轉換路徑（非常重要）
短影音不是目的，是流量入口。  

你要設計：  

抖音 / IG Reels  
→ 加LINE  
→ 免費空間照明建議  
→ 報價  
→ 成交  

🎯 免費給「燈光配置建議PDF」  
（您可以做一份專業版，這會超強）
